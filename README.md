## GoStop Game

### 고스톱 프로그램 설계

카드 구현
- 카드 객체 : 예성
- 카드 객체를 상속 받는 각각의 월별 카드
- 1월 - 4월 : 민규
- 5월 - 8월: 형기
- 9월 - 12월: 예성

플레이어 - 3인 게임
- 일단은 한 콘솔에서 3명이 플레이
- 각자 500(만원)씩 들고 시작
- 파산하면 게임 종료
- 탈주 가능 >> 남을 플레이어끼리 절반씩 나눠갖음

점수 계산기
- 사용자가 카드를 얻을 때 마다 점수 계산

승리 조건
- 1000(만원) 먼저 만들면 승리
- 다 탈주 해서 혼자 남으면 승리

게임 규칙
- 7장 손에 들고 6장은 바닥에.
- 3점 먼저 내면 이김. 고 또는 스톱 결정
- 선은 1번 플레이어가 선
